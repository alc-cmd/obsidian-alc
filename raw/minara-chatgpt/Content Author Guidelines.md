# Content Author Guidelines

## Working with AI (Midjourney, DALL-E, etc.)
- **Allowed:** Using neural networks to create individual elements (e.g., the Minara character)
- **Required:** Manually refining the result. Final work must be assembled in Photoshop, Figma, or other editors
- **Prohibited:** Publishing "raw" generations without processing — zero value, will not be considered
- **Tip:** Save screenshots of process (WIP) — best way to prove original work in disputes

## Copyright and Plagiarism
- Any copying of others' content (text or graphics) strictly prohibited
- **First violation:** Official warning + nullification of current work's value
- **Repeated violation:**
  - If you have a status role (Contributor/Creator) → immediately removed
  - If no role yet → lose opportunity to receive one (blacklist)

## Content Categories

### Graphic Content
- Profile pictures / Avatars: Unique PFPs and character art
- Stickers / Memes: Shareable reaction content
- Infographics: Visual explainers of Minara features
- Banner / Cover Art: Community banners, event headers

### Text Content
- Tutorials / Guides: How-to content for features
- Reviews / Opinions: Honest product takes
- Thread-style Breakdowns: Twitter/X threads explaining concepts
- Translations: Localizing existing English content

## Submission Format
- All submissions via designated Discord channel
- Include: Final work + WIP screenshots + brief description
- Tag relevant moderator for review
