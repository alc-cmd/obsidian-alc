# Minara Skill Installation Guide

## Value Prop
Once ClawdBot has Minara Skill installed, actions like enabling/disabling AI Autopilot can be done directly from Telegram on phone.

Minara's web features become a personal AI trading bot through OpenClaw:
- Check news, research markets, analyze trends
- Manage Autopilot
- Receive notifications anytime from Telegram
