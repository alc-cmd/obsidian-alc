# Nous Research

AI研究公司，开源 AI Agent 框架 [[Hermes Agent]] 的开发者。以加密原生方式融资，CEO Jeffrey Quesnelle 出身以太坊 MEV 基础设施项目 Eden Network。

## Key Facts
- 累计2轮融资，约 $70M，投资方为加密领域头部机构
- 融资以代币计价而非传统股权，核心用于算力储备与团队扩张
- Hermes Agent 两个月内 GitHub 47K+ stars（v0.9.0 时 53.2K）
- 官方仍处于"未发行代币"状态，但社区围绕空投预期有大量讨论
- 拥有 Atropos RL 训练框架，用于 Agent 自我改进
- 提供 Nous Portal 作为模型访问入口
